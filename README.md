# Steins;Git: A Modern Git Client

Steins;Git is a desktop Git client built with Compose for Desktop, merging powerful version control features with a sleek, thematic interface inspired by the anime Steins;Gate. Whether you're a beginner or a power user, Steins;Git combines ease of use with advanced functionality to enhance your Git experience.

---

## Key Planned Features

### 🌟 **Custom Actions Support**
- Easily implement and execute **custom Git actions**, tailored to your workflow.
- Integrated support for popular tools like **Commitizen**, streamlining your commit messages and ensuring consistency.

### ⚡ **Workflow and Batch Script Support**
- Automate repetitive tasks with **workflow scripts**.
- Manage batch operations effortlessly to save time and boost productivity.

### 🔄 **Auto-Updating Projects and Synced Repos**
- Keep all your repositories up-to-date automatically.
- Seamlessly sync changes across repositories, ensuring your work is always in sync.

### 🎨 **Modern and Intuitive UI with Steins;Gate Aesthetics**
- Immerse yourself in a Git client that pays homage to the **Steins;Gate universe**.
- The interface is designed with **dark sci-fi themes, glowing accents, and retro-futuristic fonts** reminiscent of the game.
- Dynamic elements and subtle animations echo the spirit of time travel and discovery.

---

## Why Steins;Git?

Steins;Git isn't just a Git client; it's an experience. Inspired by the **scientific curiosity and ingenuity** of the Steins;Gate series, we designed a Git client that makes Git simple for everyday tasks but robust enough for complex workflows.

With **customizable actions**, **workflow automation**, and a thematic UI, Steins;Git is your **personal Git assistant**—ensuring every commit, pull, and merge feels like part of an experiment in version control mastery.

## Get Started

1. Clone the repository:
   ```bash  
   git clone https://github.com/OmyDaGreat/SteinsGit.git  
   ```  
2. Navigate to the project directory:
   ```bash  
   cd steins-git  
   ```  
3. Build and run the app:
   ```bash  
   ./gradlew run  
   ```
